Slider – A Web Port of Animal Crossing (GameCube)

Warning: Safari and all iOS browsers have a known memory leak (WebKit bug 304810) that causes this application's memory usage to climb until the page crashes after about a minute. For a stable session try a Chrome, Edge, or any other Chromium-based browser on a desktop OS or Chrome on Android.
  1. Although this is a port and not an emulator, no assets are distributed here. A disc image file of AC is still necessary to play the game for this reason.
  2. Thank you to ACreTeam and flyngmt for their work on the decomp and PC port of ACGC, on which this project is built on top of.
  3. This software is provided as-is, without warranty of any kind. There's no guarantee it won't corrupt your save data at some point, so export your save data as often as necessary. I have not playtested the entire game, at this stage there are very likely to be places where the game crashes before you have an opportunity to save.
  4. This page is a PWA (Progressive Web App) which means you can save it locally on your device for an app-like experience with offline support. Tap your browser's Share menu and choose Add to Home Screen, Open your browser's menu and press Install / Add to Home Screen.
